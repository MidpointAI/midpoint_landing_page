# Claude Code Instructions

Use the repo-local shared skills in `skills/` for this project. The `.claude/skills/` path is linked to the same directory, so Claude Code can discover the same skill set as Codex.

When a task matches an installed skill, open the corresponding `SKILL.md` from `skills/<skill-name>/SKILL.md` and follow it. Prefer these project skills over global skills for work in this repository.
